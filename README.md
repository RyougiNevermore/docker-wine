# docker-wine

Docker Base Image for Wine

[![dockeri.co](https://dockeri.co/image/zixia/wxwork)](https://hub.docker.com/r/zixia/wxwork/)

## Features

1. `WINEARCH=i386`
1. `gecko`
1. `winetricks win7`
1. `winetricks riched20`
1. User:Group `user:group`
